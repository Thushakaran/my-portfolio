# My Portfolio

A personal portfolio website showcasing my skills, projects, and experience as a Full-Stack Software Engineer.

## 🚀 Live Demo

Visit: [https://thushakaran.github.io/](https://thushakaran.github.io/)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Animate.css
- Font Awesome

## 📋 Features

- Responsive design
- Dark/Light theme toggle
- Animated sections
- Project showcase
- Skills display
- Contact information
- Downloadable resume

## 🚀 Deployment

This portfolio is deployed using GitHub Pages.

### To deploy:

1. Install dependencies:

   ```bash
   npm install
   ```

2. Deploy to GitHub Pages:
   ```bash
   npm run deploy
   ```

### Local Development:

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start local server:
   ```bash
   npm start
   ```
